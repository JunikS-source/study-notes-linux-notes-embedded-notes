Linux được viết dựa trên UNIX
Linux là kernel – phần lõi của hệ điều hành, chịu trách nhiệm quản lý phần cứng và cung cấp system call. (quản lý CPU, RAM, device, system call)
GNU “GNU’s Not Unix” cung cấp ( gcc(compiler), glibc, bash (shell), coreutils (ls, cp, mv…) )
GNU tools + Linux kernel = OS (operate system) hoàn chỉnh
Linux distribution (hay gọi là linux distro) là Một “bộ phân phối” gồm Linux kernel + user space + cấu hình + công cụ
ubuntu thì dùng apt (Advanced Package Tool)
CentOS thì yum/dnf (Yellowdog Updater, Modified / Dandified YUM)
apt hay yum/dnf là công cụ quản lý phần mềm trong user space.
